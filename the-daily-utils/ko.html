The Daily Utils

개인정보처리방침

한국어 English 日本語 中文 Español
최종 업데이트: 2024년 9월 29일

1. 개요

더 데일리 유틸리티(The Daily Utils) 앱은 사용자의 개인정보를 수집, 저장 또는 전송하지 않습니다. 본 앱은 사용자의 프라이버시를 최우선으로 고려하여 설계되었습니다.

2. 수집하는 정보

🔒 수집하지 않는 정보

  • 개인식별정보 (이름, 이메일, 전화번호 등)
  • 연락처 정보
  • 위치 정보
  • 사진 및 동영상
  • 기기 고유 식별자
  • 사용 패턴 및 분석 데이터

3. 권한 사용 목적

📷

카메라 권한 (android.permission.CAMERA)

  • 거울 기능: 전면 카메라를 통한 실시간 미리보기 제공
  • 손전등 기능: 후면 카메라 플래시를 통한 조명 제공
  • 데이터 저장: 카메라 영상은 저장되지 않으며, 실시간으로만 사용됩니다
📳

진동 권한 (android.permission.VIBRATE)

  • 햅틱 피드백: 사용자 인터페이스 상호작용 시 진동 피드백 제공
  • 사용자 경험 향상: 터치 및 버튼 클릭 시 촉각적 반응
🔦

플래시라이트 권한 (android.permission.FLASHLIGHT)

  • 손전등 기능: 기기의 플래시 LED 제어
  • 조명 제공: 어두운 환경에서 밝은 조명 제공
⏰

웨이크 락 권한 (android.permission.WAKE_LOCK)

  • 화면 유지: 특정 기능 사용 중 화면 꺼짐 방지
  • 연속 사용: 손전등, 거울 등 연속 사용이 필요한 기능 지원

4. 데이터 저장 및 처리

🏠 로컬 저장

앱 설정 정보만 기기에 로컬로 저장됩니다

🌐 외부 전송

어떠한 데이터도 외부 서버로 전송하지 않습니다

📊 분석 데이터

사용 분석이나 추적 데이터를 수집하지 않습니다

☁️ 클라우드 백업

클라우드 서비스에 데이터를 저장하지 않습니다

5. 제3자 서비스

본 앱은 다음과 같은 제3자 서비스를 사용하지 않습니다:

  • 광고 네트워크
  • 분석 서비스
  • 소셜 미디어 연동
  • 클라우드 저장 서비스
  • 결제 처리 서비스

6. 아동 개인정보 보호

본 앱은 13세 미만 아동을 포함한 모든 연령대에 적합하며, 어떠한 개인정보도 수집하지 않으므로 아동의 개인정보 보호에 관한 특별한 조치가 필요하지 않습니다.

7. 인터넷 연결

본 앱은 인터넷 연결이 불필요하며, 완전히 오프라인으로 작동합니다. 따라서 네트워크를 통한 데이터 전송이 발생하지 않습니다.

8. 보안 조치

개인정보를 수집하지 않으므로 별도의 보안 조치가 필요하지 않으나, 앱 자체의 보안을 위해 다음과 같은 조치를 취합니다:

  • 최신 Android 보안 가이드라인 준수
  • 권한 요청의 최소화
  • 코드 난독화를 통한 앱 보안 강화

9. 정책 변경

본 개인정보처리방침에 변경사항이 있을 경우, 앱 업데이트를 통해 공지하며, 이 페이지에서 최신 정보를 확인할 수 있습니다.

10. 문의

개인정보처리방침에 대한 문의사항이 있으시면 앱 내 설정 페이지를 통해 연락해 주세요.
